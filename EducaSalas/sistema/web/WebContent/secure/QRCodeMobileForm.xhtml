PRIMEIRO ACESSO

Para acessar a sua conta no aplicativo móvel ADMRH, é preciso informar o Órgão/Empresa que você está vinculado. Faça isso utilizando uma das opções abaixo:

LEITURA DO QR-CODE

Abra o opção de fazer a leitura do QR-Code e aponte a câmera para o código QR abaixo

DIGITAÇÃO DO CÓDIGO

Na tela de digitação do código do Órgão/Empresa, informe o código abaixo:

#{ qRCodeMobileBacking.codigoOrgaoEmpresa }
